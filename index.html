以筆劃邏輯打造更直覺的中文輸入法

StrokeIME 是一套以「橫、豎、撇、點、折」為核心的中文筆劃輸入法。 使用者只要依照中文字的書寫順序輸入指定按鍵，系統便會搜尋字庫並顯示候選字， 讓中文輸入更簡單、更直覺，也更適合專題展示與教學應用。

About Project

專題介紹

降低中文輸入門檻，讓使用者不用熟悉注音、拼音或倉頡字根，也能透過中文字形與筆劃順序完成輸入。

目

專題目標

建立一套專屬中文筆劃輸入法，讓使用者透過簡單按鍵輸入筆劃，再由系統找出符合條件的中文字。

邏

核心概念

將中文字拆解為筆劃編碼，透過字串比對與候選字清單，完成從筆劃輸入到中文字輸出的流程。

用

應用價值

可應用於中文教育、查字工具、輸入輔助系統，也適合不熟悉發音輸入法的使用者使用。

System Flow

系統流程展示

從使用者輸入筆劃，到系統搜尋候選字，再完成選字輸入，形成完整的輸入法操作流程。

輸入流程

系統會即時接收使用者的鍵盤輸入，並依照筆劃編碼進行查詢。

1使用者輸入 u、i、o、j、k 或 l
2程式將按鍵轉換為中文筆劃序列
3比對字庫中符合條件的中文字
4顯示候選字並支援數字鍵選字

操作範例

使用者不確定完整筆劃時，也能透過 l 作為萬用字元進行模糊查詢。

u代表第一筆為「一」
l代表第二筆可以是任意筆劃
ul查詢開頭符合「橫 + 任意筆劃」的候選字
選使用 0、9、8…2、1 或方向鍵選擇文字
Features

功能特色

以「簡單、直覺、可操作」為主要設計方向，讓使用者能快速理解並實際使用。

筆劃輸入

使用 u、i、o、j、k 對應五種基本筆劃，依照中文字書寫順序輸入即可查找文字。

萬用字模糊搜尋

使用 l 作為萬用字元，當使用者不確定某一筆時，可用 l 代替未知筆劃。

候選字顯示

系統會根據輸入的筆劃序列，列出所有符合條件的中文字候選清單。

快速選字與翻頁

候選字可使用數字鍵選取，也能透過上下鍵移動選擇或翻頁，提高輸入效率。

刪除已選文字

支援 Ctrl + Backspace 刪除已選好的中文字，方便使用者修正輸入錯誤。

電腦端試用

How To Use

操作方式

簡單記住六個按鍵，即可開始使用筆劃輸入法。

1

輸入筆劃

依照中文字的書寫順序輸入對應按鍵，例如「一」可輸入 u，「二」可輸入 uu。

2

查看候選字

系統會顯示符合筆劃條件的中文字，讓使用者從候選字列表中選擇目標字。

3

選字完成輸入

可使用數字鍵快速選字，也可使用方向鍵移動選擇與翻頁，完成輸入流程。
